## Design Decision System

Identify the product category:
- SaaS
- Dashboard
- E-commerce
- Portfolio
- Mobile app
- Landing page
- Admin panel
- etc.

---

### 2. Style Selection
Choose an appropriate UI style based on product type:

Examples:
- SaaS → clean / minimal / modern
- E-commerce → visual / promotional
- Dashboard → structured / data-focused
- Portfolio → expressive / creative

---

### 3. Color System Definition

Define:
- Primary color
- Secondary color
- Background
- Accent colors

Ensure:
- Good contrast
- Accessibility
- Visual consistency

---

### 4. Typography System

Select:
- Heading font
- Body font

Ensure:
- Readability
- Hierarchy
- Consistency

---

### 5. Layout Strategy

Always structure UI into:

- Header / Navigation
- Main content sections
- Key components (cards, lists, etc.)
- CTA (Call To Action)
- Footer (if applicable)
